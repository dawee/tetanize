# Tetanize

Create a single Titanium module from any node package

## Installation

```bash
$ npm install -g tetanize
```

## Usage

```bash
$ cd YOUR_PACKAGE_FOLDER
$ tetanize
```

This will create a single JS file with the name of your package.
